Автоматизация теста к API

Для запуска тестов должны быть установлены пакеты pytest и requests
Запуск всех тестов выполняется командой pytest Задание: Клиент создает заказ. Проверяется, что по треку заказа можно получить данные о заказе. Шаги автотеста: Выполнить запрос на создание заказа. Сохранить номер трека заказа. Выполнить запрос на получение заказа по треку заказа. Проверить, что код ответа равен 200.
